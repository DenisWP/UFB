Para executar o projeto, deverá ser criado uma tabela "ufb". 

Dentro dela, criar tres tabelas:

- administrator
- candidatos 
- cursos

Os scripts SQL com os campos de cada tabela estão na pasta "Scripts". 


ATENÇÃO

Os cursos ofertados pela UFB que terão vínculo com as páginas "cursos", são apenas os apresentados no script SQL. Nada impede do usuário
cadastrar novos cursos, porém, não será criada uma página para eles, pois não estaram referenciados na página php. (As páginas deverão ser 
criadas manualmente) 

As demais páginas, vestibular, cadastrar curso, excluir curso, editar curso ... todas estão interligadas ao banco de dados.
